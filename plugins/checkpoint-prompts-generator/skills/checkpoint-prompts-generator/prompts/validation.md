# 文件验证失败提示模板

## 文件格式检查失败

```
⚠️ 文件格式检查发现问题：
• 问题：[具体问题描述]
• 位置：[文件路径/第X行]

请选择处理方式：
1) 让Skill尝试自动修复（会先备份原文件）
2) 中止操作（请自行修复后重新提供文件）
```

常见问题包括：
- 文件不存在
- 文件类型不支持（仅支持Excel/CSV）
- 列数不足7列
- B-G列数据为空
- 数据内容不像审查数据

## 必需字段缺失

```
⚠️ 文件验证失败 - 缺少必需字段：
• 缺失字段：[字段名]
• 问题位置：[相关单元格/行列]

请选择处理方式：
1) 让Skill尝试自动修复（会先备份原文件）
2) 中止操作（请自行修复后重新提供文件）
```

审查数据必需字段：
- 审查环节
- 审查点名称
