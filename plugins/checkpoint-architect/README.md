# checkpoint-architect

审查点构建助手 - 帮助产品经理通过引导式对话创建符合规范的审查点内容。

## 功能特性

- 引导式对话，无需技术背景
- 两种模式：快速模式（自然语言描述）+ 引导模式（逐步提问）
- 5 种审查模式：文书完整性、时限合规性、信息一致性、程序合规性、自定义逻辑
- 自动验证内容格式
- 生成可视化流程图（Mermaid + ASCII）
- 自动推断推荐信息点清单，帮助了解需要从各文书中抽取的关键字段
- 输出文件包含免责声明，提醒人工核实

## 使用方式

安装后，在 Claude Code 中输入 `/checkpoint-architect` 即可启动。

## 文件结构

```
skills/checkpoint-architect/
├── SKILL.md                  ← 主流程（9 个步骤）
├── checkpoint-templates.md   ← 5 种审查模式的模板和引导问题
└── checkpoint-validator.md   ← 内容格式验证规则
```

## 输出格式

生成的审查点文件保存至 `output/YYYY-MM-DD-{审查点名称}.md`，包含：

- 审查步骤（第一步/第二步/第三步）
- 涉及文书类型
- 推荐信息点清单
- 免责声明
