# Claude Marketplace

BreezeWang 的 Claude Code 插件市场。

## 插件列表

| 插件 | 说明 |
|------|------|
| [checkpoint-architect](plugins/checkpoint-architect/) | 审查点构建助手，引导式对话创建审查点内容 |
| [checkpoint-prompts-generator](plugins/checkpoint-prompts-generator/) | 审查要点Prompt生成器，从Excel文件生成法律审查长提词 |
| [iflytek-uap](plugins/iflytek-uap/) | UAP 统一认证平台集成助手 |

## 安装方法

```bash
# 添加市场
/plugin marketplace add breezewang19-lang/claude-marketplace

# 安装插件
/plugin install checkpoint-architect@yrwang45-marketplace
/plugin install checkpoint-prompts-generator@yrwang45-marketplace
/plugin install iflytek-uap@yrwang45-marketplace
```
